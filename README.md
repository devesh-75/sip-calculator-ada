# SIP Investment Calculator

An interactive SIP calculator built to improve investor education.

## Tech Stack
- Next.js
- Node.js
- MySQL
- Drupal CMS
- PHP

## Features
- Interactive sliders
- Investment breakdown chart
- SIP future value calculation
- Responsive UI
- CMS integration

## Architecture
User → Drupal CMS → Next.js Calculator → Node API → MySQL
